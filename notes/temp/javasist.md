





| 符号                  | 含义                                            |
| --------------------- | ----------------------------------------------- |
| `$0`, `$1`, `$2`, ... | `this` and 方法的参数                           |
| `$args`               | 方法参数数组.它的类型为 `Object[]`              |
| `$$`                  | 所有实参。例如, `m($$)` 等价于 `m($1,$2,`...`)` |
| `$cflow(`...`)`       | `cflow` 变量                                    |
| `$r`                  | 返回结果的类型，用于强制类型转换                |
| `$w`                  | 包装器类型，用于强制类型转换                    |
| `$_`                  | 返回值                                          |
| `$sig`                | 类型为 java.lang.Class 的参数类型数组           |
| `$type`               | 一个 java.lang.Class 对象，表示返回值类型       |
| `$class`              | 一个 java.lang.Class 对象，表示当前正在修改的类 |
